Frontend

HTML/CSS/JavaScript HTML로 웹페이지의 구조를 작성하고, CSS를 통해 스타일을 지정하며, JavaScript로 동적인 기능을 구현. 예시 프로젝트: 학내 프로젝트: HTML/CSS/JS를 사용한 웹 디자인 프로젝트. UI/UX 디자인 프로젝트: 사용자의 상호작용을 고려한 동적인 웹페이지 설계. UI/UX 디자인 웹 페이지 레이아웃 구성 및 사용자 경험 중심의 인터페이스 설계. Visual Studio Code와 같은 툴을 사용해 기초부터 발전된 UI/UX 디자인 학습. 프론트엔드 프레임워크 및 도구 jQuery: DOM 조작 및 Ajax를 활용한 효율적인 프론트엔드 개발. 연동 및 동적 기능 구현 MariaDB와 MySQL 연동으로 데이터 기반 웹 애플리케이션 구현. 동적인 웹 요소와 실시간 데이터 업데이트 기능 개발.

Backend

프로그래밍 언어 Java: 객체지향 프로그래밍, 데이터베이스 연동, Java Swing으로 데스크톱 애플리케이션 개발. Python: 알고리즘 설계, 데이터 처리, 간단한 서버 애플리케이션 구축. 데이터베이스 MySQL/MariaDB: 관계형 데이터베이스 설계, 쿼리 작성 및 연동. 데이터베이스 응용 프로젝트 진행 경험: 쇼핑몰 데이터베이스 설계 및 구축. KPOP MUSIC STORE 앱 데이터 연동. 서버 관리 Linux: 기본적인 서버 환경 구성 및 관리. VMware: 가상 서버 시스템 실습 및 관리 경험. 웹 백엔드 개발 PHP: 간단한 서버 스크립트 작성. Java Servlet/JSP: 웹 애플리케이션의 서버 측 로직 개발. 보안 및 네트워크 Kali 및 Metasploit 환경에서 보안 실습 경험. 서버와 클라이언트 간의 안전한 데이터 통신 구조 이해.

웹 퍼블리셔

웹 퍼블리셔는 단순히 코드로 화면을 구현하는 것을 넘어, **사용자 경험(UI/UX)**과 디자인의 완성도를 고려하여 웹사이트를 구축하는 직업입니다.

강점: 디자인과 개발을 아우르는 다학제적 역량 UI/UX를 고려한 인터랙티브한 웹 구현 협업을 통해 창의적인 아이디어를 현실로 구현

저는 이러한 강점을 바탕으로, 사용자 중심의 디자인과 효율적인 개발을 통해 높은 완성도의 웹사이트를 구축하는 것을 목표로 하고 있습니다.
